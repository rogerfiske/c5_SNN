The dataset 'CA5_date.csv' has 6 cols:
date,m_1,m_2,m_3,m_4,m_5
2/4/1992,5,8,10,30,38
The file contains a record of part numbers used (not number of parts) in 5 machines at the same location and operating under identical conditions. Where cols; m_1,m_2,m_3,m_4,m_5 are the part numbers (1-39 inclusive) that are required for each of the 5 identical machines running 18 hours/day since installation 2/4/1992. No duplicate part numbers can be shipped on a given day and 5 parts must be shipped.
